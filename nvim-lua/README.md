# Useful keybindings and commands

## Keybindings

### Filesystem tree (Neo-Tree)

Ctrl + n : print or close folder filesystem tree

## Commands
### Markdown (peek.nvim) 
:PeekOpen : Open a real time markdown preview

### Git (vim-fugitive)
:Git commit  
:Git diff  
:Git blame  
:Git  

:Gdelete : Equivalent to "git rm"
